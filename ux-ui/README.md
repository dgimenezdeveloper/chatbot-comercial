# UX/UI — chatbot-comercial 🎯

Este directorio contiene los recursos y la documentación relacionados con la experiencia de usuario (UX) y la interfaz de usuario (UI) del proyecto **chatbot-comercial**.

## 📁 Contenido

| Directorio     | Descripción                                        |
|----------------|----------------------------------------------------|
| `/assets`      | Imágenes, íconos y recursos de diseño              |
| `/wireframes`  | Bocetos y diagramas de flujo de interacción        |
| `/mockups`     | Prototipos visuales en alta y baja fidelidad       |
| `/guidelines`  | Guías de estilo, tipografías y paleta de colores   |

## 🏗️ Estructura Sugerida

```bash
ux-ui/
├── README.md        # Este archivo
├── assets/          # Recursos gráficos
├── wireframes/      # Bocetos y flujos
├── mockups/         # Prototipos visuales
└── guidelines/      # Guías de estilo
```

## ⚙️ Instalación / Apertura de Archivos

1. **Clona el repositorio:**

   ```bash
   git clone https://github.com/Jann-CH/chatbot-comercial.git
   cd chatbot-comercial/ux-ui
   ```

2. **Abre los archivos de diseño** con tu herramienta preferida (Figma, Sketch, Adobe XD, o visualizador de imágenes).

## 🎨 Uso

- Utiliza los **wireframes** para entender los flujos de usuario.
- Refiérete a los **mockups** para la implementación visual.
- Sigue las **guidelines** para mantener consistencia en tipografías, colores y componentes.

## 🤝 Contribuir

- Describe cambios en un **issue** antes de trabajar en un diseño importante.
- Crea ramas con el prefijo `ux/` o `ui/` y abre un **pull request** cuando esté listo.
- Incluye capturas o enlaces a prototipos en el PR.

## ✅ Recomendaciones de Diseño

- **Accesibilidad:** contraste adecuado, tamaños táctiles y navegación por teclado.
- **Mobile-first:** diseñar pensando primero en dispositivos móviles.
- **Componentes reutilizables:** tokens de diseño y biblioteca de componentes.
